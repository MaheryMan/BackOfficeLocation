=========================
UTILISATION API PROTÉGÉES
=========================
Toutes les requêtes vers /api/reservations/* nécessitent un paramètre token:

Exemples:
- GET  /api/reservations?token=xxx
- GET  /api/reservations/1?token=xxx
- POST /api/reservations?token=xxx&idClient=1&idHotel=2&...
- POST /api/reservations/1?token=xxx&idClient=1&...
- POST /api/reservations/1/delete?token=xxx

Pour obtenir un token:
- GET /api/token (défaut: 7 jours de validité)
- GET /api/token?days=30 (validité personnalisée)

Pour générer les tokens initiaux:
- Exécuter: java util.GenerateTokensMain
